## `Map` 인터페이스

- 순서 없음, 데이터 중복은...
    - 키(key; キー)는 중복되면 안 되나 값(value; 値)은 중복이 가능
- 구현: `HashMap<K, V>`
    - 키와 값의 쌍으로 구성되는 요소를 다룸
    - 값을 얻기 위해선 반드시 키를 이용해야 함
