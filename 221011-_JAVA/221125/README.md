## Oracle SQL Developer 소프트웨어 실습

```sql
INSERT INTO STUDENT (
    NUM,
    NAME,
    ADDRESS,
    TEL,
    KOR,
    ENG,
    MAT
) VALUES (
    1111,
    '케이드',
    '대전광역시',
    '010',
    100,
    100,
    100
);
```

- 컬럼이 7개인 테이블에 데이터를 7종류 입력하는 그런 것.
- `WHERE`
    - `OR`와 `AND`로 조건의 범위를 결정
    - `LIKE`로 `VARCHAR2` 타입 레코드에 있는 데이터 범위 특정
